# 올인원 변환 도구 프로젝트 요약

## 📋 프로젝트 개요

**프로젝트 명:** ConvertHub (올인원 변환 도구 사이트)

**목표:**
- 무료 파일 변환/유틸리티 도구 제공
- Google AdSense로 광고 수익 창출
- 점진적 기능 추가 (하나씩 완성)
- 첫 배포 프로젝트로 실전 경험 쌓기

**타겟:**
- 파일 변환이 필요한 일반 사용자
- SEO 트래픽 기반 무료 방문자 유치

---

## 🛠️ 기술 스택 결정

### 프론트엔드
- **프레임워크:** React + TypeScript (또는 Next.js)
- **스타일링:** Tailwind CSS
- **빌드 도구:** Vite 또는 Next.js
- **배포:** Cloudflare Pages (무료)

### 백엔드
- **프레임워크:** FastAPI (Python)
- **이유:**
  - Spring Boot보다 가벼움 (콜드 스타트 빠름)
  - 무료 호스팅 옵션 좋음
  - 현대적이고 배우기 쉬움
  - Spring Boot 개념과 유사 (DI, REST API)
- **배포:** Railway (무료 티어 $5 credit) 또는 Render

### 데이터베이스
- 초기에는 불필요
- 필요 시: PostgreSQL (Supabase 무료 티어)

```
┌─────────────────────────┐

            │

            │
            ▼

│  (Python)               │
└─────────────────────────┘
```

### 처리 방식 분리

**클라이언트 사이드 (브라우저에서 처리):**
- 이미지 변환/압축 ⭐ (첫 기능)
- 이미지 리사이즈
- QR 코드 생성
- 텍스트 도구 (대소문자, 글자수)
- JSON 포맷터
- Base64 인코딩
- 색상 변환

**서버 사이드 (FastAPI 필요):**
- PDF 변환 (이미지 → PDF)
- PDF 병합/분할
- 배경 제거 (AI 처리)
- OCR (광학 문자 인식)
- 복잡한 파일 변환

## 🎯 첫 번째 기능: 이미지 변환/압축

### 선택 이유
✅ 클라이언트 사이드로 구현 가능 (서버 비용 없음)
✅ 검색량 높음 (SEO 유리)
✅ 개발 기간 짧음 (1-2일)
✅ 빠른 성취감
✅ 재방문율 높음

### 기능 상세
1. **이미지 포맷 변환**
   - JPG ↔ PNG ↔ WebP 변환
   - 브라우저에서 직접 처리 (Canvas API)

2. **이미지 압축**
   - 품질 조절 가능
   - 파일 크기 감소
   - 미리보기 제공

3. **이미지 리사이즈**
   - 폭/높이 지정
   - 비율 유지 옵션
   - 일반 크기 프리셋 (SNS용 등)

### 사용 라이브러리
- `browser-image-compression` (압축)
- Canvas API (변환/리사이즈)
- React Dropzone (파일 업로드 UI)
